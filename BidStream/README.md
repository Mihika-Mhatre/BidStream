# BidStream

BidStream is a live bidding and auction platform scaffolded with a Java backend, MongoDB persistence, and a fintech-themed frontend.

## Structure

- `backend/` - Spring Boot backend with MongoDB models and service architecture.
- `frontend/` - Declarative HTML, CSS, and event-driven JavaScript for live auction interactivity.
- `config/mongodb/` - MongoDB schema definitions and collection setup.
